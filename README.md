# AyurSutra – Panchakarma Management Platform
## Overview  
AyurSutra is a web-based platform designed to simplify and digitize Panchakarma therapy management. It provides a centralized system for patients and practitioners to manage therapy schedules, track progress, and improve overall treatment efficiency.

## Problem  
Panchakarma centers often rely on manual processes, which lead to:
- Inefficient appointment scheduling  
- Unorganized patient records  
- Difficulty in tracking therapy progress  
- Lack of structured communication  

## Solution  
AyurSutra introduces a digital platform that enables:
- Automated therapy scheduling  
- Centralized patient management  
- Role-based dashboards for patients and practitioners  
- Secure authentication and data handling  

## Features  
- User authentication  
- Therapy scheduling system  
- Patient and practitioner dashboards  
- Patient data management  
- Responsive interface  

## Tech Stack  
- React (Vite)  
- TypeScript  
- Tailwind CSS  
- shadcn/ui  
- Supabase
